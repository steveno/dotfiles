This is a saltstack setup that I use to manage my system along with
my dotfiles. The following files are currently being managed:

- [bash_aliases](./files/bash_aliases) - bash configuration
- [gitconfig](./files/gitconfig) - global gitconfig
- [login.sql](./files/login.sql) - for Oracle's SQLPlus
- [profile.ps1](./files/powershell/profile.ps1) - for Powershell on Windows
- [scripts.ps1](./files/powershell/scripts.ps1) - companion scripts for profile.ps1
- [init.vim](./files/vime/init.vim) - neovim configuration
